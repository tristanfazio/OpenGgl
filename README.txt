~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

main.cpp - main program handler
ground.cpp - object to draw ground squars
trees.cpp - object to draw trees throughout the scene
wall.cpp - draws the fence posts/walls and sliding door/wall
sven.cpp - draws the sven dog character
watersheep.cpp - draws the evil watersheep sheep character
torch.cpp - draws the objective light source and animation
utils/textureloader.cpp - handles the loading of textures for display
utils/stb_image.c - used by texture loader to aid in the loading of images

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To Build:

	"make"
To Run:
	"./main"

Depending on whether OS is mac or linux, the MACLIB/LINUXLIB value may need to be swapped in makefile
